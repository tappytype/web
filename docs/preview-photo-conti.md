# preview "일상 곳곳에" 활용샷 촬영 콘티

> 목적: preview 섹션의 카드를 **완성된 내 폰트(.ttf)가 실제로 쓰인 사진**으로 채우기 위한 촬영 가이드.
> 현재는 캘린더 일러스트 + 앱 실사진 2장. 아래 컷을 찍으면 일러스트를 진짜 활용샷으로 교체하거나 카드를 늘릴 수 있다.

## 단 하나의 판별 기준

**"이 사진에 내가 만든 손글씨 글씨체가 실제로 찍혀 있는가?"**
- 있으면 → 활용샷 (preview에 들어갈 자격)
- 없으면 → 분위기샷 (preview에는 부적합)

그래서 모든 컷의 핵심은 "내 폰트로 쓴 글자가 프레임 안에 또렷이 보이게"다.

## 공통 촬영 스펙

- **비율: 3:4 세로** (카드가 `aspect-ratio:3/4`, `object-fit:cover`라 세로로 찍어야 안 잘림)
- **해상도**: 긴 변 1600px 이상 (가공해서 821×1095로 줄일 거라 여유 있게)
- **조명**: 창가 자연광, 따뜻한 톤. 직사광·형광등 그림자 피하기
- **색감**: 크림/블러시 핑크/라벤더 소품이 사이트 팔레트(`#FFF7F3`, `#FFE4EA`)와 맞음
- **여백**: 폴라로이드 프레임 + 살짝 회전(-1.2°/+1.4°)이 자동 적용되니, 가장자리에 핵심을 두지 말고 중앙에 여유 있게
- **글자 가독성**: 내 폰트로 쓴 텍스트가 화면에서 작아도 "손글씨구나" 알아볼 정도 크기로

## 컷 A — 아이패드 캘린더 실사용 ⭐ (1순위, 캘린더 일러스트 대체용)

원래 원했던 "진짜" 버전. 일러스트를 이걸로 교체.

```
┌─────────────────┐ 3:4
│  ☀ 창가 자연광   │
│ ┌─────────────┐ │
│ │ 6월 캘린더    │ │  ← 캘린더 앱 / 굿노트 월간 뷰
│ │ ┌─┬─┬─┬─┐   │ │
│ │ │약속│  │시험│ │ │  ← 각 칸 일정을 *내 폰트*로 입력
│ │ ├─┼─┼─┼─┤   │ │
│ │ │  │카페│  │  │ │
│ │ └─┴─┴─┴─┘   │ │
│ └─────────────┘ │
│   ✏️ 펜슬 옆에    │
└─────────────────┘
```

- **찍는 것**: 아이패드에 캘린더 앱(또는 굿노트 월간 템플릿)을 띄우고, 일정 텍스트 몇 개를 **내 손글씨 폰트로** 입력한 화면
- **구도**: 책상 위 아이패드를 약간 위에서 비스듬히(15~25°). 애플펜슬을 캘린더 옆에 자연스럽게
- **핵심**: 캘린더 칸 안의 일정 글자가 내 폰트일 것 (시스템 폰트면 의미 없음)
- **소품**: 우드 스탠드, 라벤더 펜슬, 메모지 — 현재 `use-app` 실사진과 톤 맞추면 좋음

## 컷 B — 굿노트 다이어리 필기

```
┌─────────────────┐ 3:4
│ ┌─────────────┐ │
│ │ 2026.6.19   │ │  ← 날짜·제목 내 폰트
│ │ ─────────── │ │
│ │ 오늘은 ...   │ │  ← 일기 본문 내 폰트
│ │ 폰트 만들었다 │ │
│ │ 너무 신기해   │ │
│ │       🩷     │ │
│ └─────────────┘ │
│  손 일부 / 펜슬   │  ← 쓰는 중인 느낌
└─────────────────┘
```

- **찍는 것**: 굿노트/노타빌리티 다이어리 페이지에 내 폰트로 일기·메모를 쓴 화면
- **구도**: 정면에서 약간 기울여. 손이나 펜슬 끝이 살짝 들어오면 "쓰는 중" 생동감
- **핵심**: 본문 텍스트가 내 폰트. 손글씨 느낌이 가장 잘 사는 컷

## 컷 C — 아이폰 잠금화면 / 위젯

```
┌─────────────────┐ 3:4
│  ┌───────────┐  │
│  │   9:41     │  │
│  │  6월 19일   │  │  ← 잠금화면 폰트를 내 글씨체로
│  │            │  │     (또는 위젯에 내 폰트 문구)
│  │ "오늘도 딸깍"│  │
│  │            │  │
│  └───────────┘  │
│   손에 든 폰      │  ← 한 손으로 든 구도
└─────────────────┘
```

- **찍는 것**: 아이폰 잠금화면 시계/날짜 폰트나 위젯 문구를 내 폰트로 설정한 화면
- **구도**: 손에 들고 살짝 위에서. 배경은 단색 벽/책상
- **핵심**: 시계·날짜·문구가 내 폰트. "매일 보는 화면에 내 글씨"라는 메시지

## 컷 D — 종이 엽서 / 손편지 (아날로그)

```
┌─────────────────┐ 3:4
│   ✉ 크래프트 봉투 │
│ ┌─────────────┐ │
│ │  To. 보배    │ │  ← 내 폰트로 인쇄/출력한 엽서
│ │             │ │
│ │ 고마워 ...   │ │
│ │      - 지인  │ │
│ └─────────────┘ │
│  말린꽃 / 마테    │  ← 따뜻한 소품
└─────────────────┘
```

- **찍는 것**: 내 폰트로 쓴 문구를 프린트한 엽서/카드를 소품과 함께
- **구도**: 위에서 내려다보는 플랫레이(flat-lay)
- **핵심**: "디지털 폰트가 종이로도" — .ttf의 활용 범위를 보여줌

## 우선순위 & 카드 운영

1. **컷 A(캘린더 실사용)** 하나만 찍어도 → 1번 캘린더 일러스트를 진짜 사진으로 교체 (가장 임팩트)
2. A + B 찍으면 → 카드 3장(캘린더 실사용 + 굿노트 + 앱 실사진)으로 확장, `.gallery`를 3열로 되돌림
3. A~D 다 모이면 → "곳곳에"라는 카피가 진짜가 됨. 4장은 2×2 그리드 권장

## 못 찍는 컷은 Google Flow 프롬프트로

실사 촬영이 어려운 컷(예: 깔끔한 캘린더 화면)은 아래 톤으로 생성 가능:

> A soft 3D claymation-style cute tiger plushie character sitting at a cozy pastel desk, using an iPad showing a **monthly calendar app with handwritten Korean schedule entries** written in a warm handmade handwriting font. Cream and blush-pink color palette (#FFF7F3, #FFE4EA), lavender Apple Pencil, warm window light, K-culture room posters softly blurred in the background, vertical 3:4 composition, high detail, adorable and clean.

핵심: **calendar app with handwritten Korean entries**(손글씨로 채워진 일정)를 명시해야 "폰트 활용"이 드러난다.
