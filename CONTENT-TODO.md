# TappyType 랜딩 — 내가 채울 콘텐츠 체크리스트

리디자인은 끝났고, 아래는 **석지인 님이 직접 만들어 와야 하는 콘텐츠**예요.
자리(placeholder)와 마크업은 이미 준비돼 있어서, 파일만 주면 끼우면 됩니다.

---

## 1. Preview 갤러리 — 실사용 사진 ⭐ (우선순위 높음)

**섹션**: "완성된 내 폰트가, 일상 곳곳에" (`#preview`)

- **무엇**: 내가 만든 손글씨 폰트가 **실제로 적용된** 예쁜 화면/사진
  (지금 1번 칸은 태피 일러스트 렌더로 *임시* 채워둠 — 진짜 샷 오면 교체)
- **몇 장**: 3~4장
- **각 칸 주제(캡션)**:
  1. 굿노트 다이어리  ← 지금 임시 렌더
  2. 프로크리에이트 레터링
  3. 키노트 슬라이드
  4. 손편지·카드
- **규격**:
  - 비율 **4:3 권장** (가로형, `object-fit:cover`로 잘릴 수 있음)
  - 폭 **~1200px**, `.jpg`, 1장당 **300KB 이하** (내가 압축해줄 수도 있음)
- **파일명(권장)**: `assets/preview-goodnotes.jpg`, `preview-procreate.jpg`, `preview-keynote.jpg`, `preview-letter.jpg`
- **팁**: 밝고 깔끔한 배경, 내 폰트가 잘 보이게. 실제 앱/굿노트 캡처가 제일 설득력 있음.
- **주면**: 내가 자리에 끼우고 모바일 노출 개수(현재 2개)도 조정.

> 참고: 모바일은 지금 앞 2칸만 보이게 해뒀어요(빈 구간 단축). 사진 다 들어오면 4칸 다 켤 수 있음.

## 2. How To — 단계 클립 (중간 우선순위)

**섹션**: "설치하고, 쓰고, 딸깍하고, 내보내면 끝" (`#howto`)

- 지금 **55초 데모 영상**은 이미 임베드됨(유튜브). 그 아래 4단계 스토리 카드 중:
  - **3번 "AI 딸깍"** → 현재 "✨ 곧 공개" placeholder
  - **4번 ".ttf 내보내기"** → 현재 "📤 곧 공개" placeholder
- **무엇**: 짧은 화면 녹화(5~10초) 또는 스크린샷 1장
- **규격**: 세로 **9:16**(스토리 카드 비율). 영상이면 `.mp4`(무음·루프 권장), 정지컷이면 `.png`/`.jpg`
- **파일명(권장)**: `assets/how-aitap.mp4`(or `.png`), `assets/how-export.mp4`(or `.png`)
- 1·2번 단계는 기존 앱 스크린샷(`shot-onboarding`, `shot-main`)으로 이미 채워져 있음.

## 3. (선택) 인스타그램 채널

랜딩과 무관하지만 같이 정리하면 좋은 것:

- `design/references/insta-1.jpeg`, `insta-2.jpeg` = 인스타 피드/하이라이트 키트 (랜딩과 동일 IA)
- `Tiger_typing_on_iPad`(한글 매직·딸깍), `Tiger_..._strawberry`(화장대) 렌더 = **인스타 게시물용**으로 좋음
- ⚠️ 인스타 키트 카피에 아직 **"Join Waitlist / coming soon"**(출시 전)이 남아 있음 → **출시 카피로 갱신 필요**

## 4. 배포

- 현재 변경사항은 **`main`에 커밋 완료**. `git push` 하면 **Cloudflare Pages가 tappytype.com에 자동 배포**.
- 푸시 전 확인: 로컬 미리보기 `python3 -m http.server` → `http://localhost:8753/`
